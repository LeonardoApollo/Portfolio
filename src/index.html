Про меня

Меня зовут Михаил

Жизнь котов тяжела, и чтобы получать больше еды от глупых человеков, великому мне, пришлось начать работать из дома. И первым делом мой зоркий глаз пал на профессию программиста. К своим 20 годам я принял решение изучать программирование самостоятельно, где-то по платным курсам, где-то по урокам из YouTube. И на этом сайте вы сможете ознакомиться с моими успехами, как кота-самоучки, что решил зарабатывать на корм себе сам.

Web-разработка
Хотя у меня отсутствует опыт реальных проектов, но каждый с чего-то начинал! Самостоятельное изучение основных технологий fontend разработки + необходимый backend для поднятия сайта на протяжении полу-года.
Разработка приложений
В настоящее время, большенство людей не предстовляют свою жизнь без телефона. И по этому необходимость в людях умеющих делать приложения высока как никогда. Я не ленился и также изучал весь инструментарий необхдимый для разрабоки приложений.
Личные качества
Самостоятельное изучение новой профессии очень трудоемкий и выматывающий процесс. Ведь тебя никто не подтолкнет и не заставит этим заниматься, как например в школе. Это очень сильно закаляет выдержку и делает целеустремленным.

Навыки

Что я использую в работе

HTML

Именно он создает каркас вашего сайта или приложения, а пятая версия позволит мне создавать более SEO-оптимизированную структуру вашего продукта

CSS3

Этот язык стилей позволяет мне создавать абсолютно любой внешний вид вашего сайта или приложения. Все ограничивается только вашей фантазией!

Java Script

Этот язык программирования позволяет оживить все что угодно: слайдеры, окна, подсказки, вкладки, получение данных от сервера и многое другое

Jqurey

Библиотека Jquery позволит ускорить разработку. Без необходимости интегрировать в проект мы её не будем, но навык работы с ней присутствует

React

Эта библиотека позволяет создавать web-приложения. Мы можем создать максимально интерактивный продукт именно под ваши цели

Node.js

Эта платформа позволяет создавать бэкенд для вашего продукта - “мозги”, которые будут выполнять действия, которые пользователь не видит

PHP

Это язык программиования общего назначения, специально сконструированный для вэб-разработок с возожностью его внедрения непосредственно в HTML

MySQL

Этот самая распространенная система управения базами данных. С помощью нее можно лего структурировать данные сервера, а также использовать для хранения данных поступающих на сервер
